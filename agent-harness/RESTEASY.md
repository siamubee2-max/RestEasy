# RestEasy Agent Harness — SOP

## Overview

This directory contains the **RestEasy CLI agent harness** — a Python CLI tool that
allows AI agents (and developers) to interact with the RestEasy TCC-I program
entirely offline, without needing the mobile app or Supabase.

                                                                                                                                                                                                                                                                  ## Quick Start
                                                                                                                                                                                                                                                                  
                                                                                                                                                                                                                                                                  ```bash
                                                                                                                                                                                                                                                                  # Start REPL
                                                                                                                                                                                                                                                                  resteasy repl
                                                                                                                                                                                                                                                                  
                                                                                                                                                                                                                                                                  # Compute sleep efficiency
                                                                                                                                                                                                                                                                  resteasy sleep efficiency --tib 480 --tst 420 --latency 15 --wakes 2
                                                                                                                                                                                                                                                                  
                                                                                                                                                                                                                                                                  # Start an offline journey
                                                                                                                                                                                                                                                                  resteasy journey new -o my_journey.json --name "Alice"
                                                                                                                                                                                                                                                                  
                                                                                                                                                                                                                                                                  # Simulate 6 weeks
                                                                                                                                                                                                                                                                  resteasy journey simulate my_journey.json --profile improving
                                                                                                                                                                                                                                                                  
                                                                                                                                                                                                                                                                  # Run tests
                                                                                                                                                                                                                                                                  pytest cli_anything/resteasy/tests/ -v
                                                                                                                                                                                                                                                                  ```

                                                                                                                                                                                                                                                                  ## Journey Module
                                                                                                                                                                                                                                                                  
                                                                                                                                                                                                                                                                  The `journey` command group simulates the complete TCC-I 6-week program offline:
                                                                                                                                                                                                                                                                  
                                                                                                                                                                                                                                                                  - Daily morning journal with automatic SE/TIB/TST/WASO calculation
                                                                                                                                                                                                                                                                  - Streak tracking with break detection
                                                                                                                                                                                                                                                                  - 12 badges (first_entry, streak_3/7/14/21/42, efficiency_85/90, etc.)
                                                                                                                                                                                                                                                                  - ISI questionnaire (7 questions, 0-28 score, 4 severity levels)
                                                                                                                                                                                                                                                                  - Weekly review with TCC-I window adjustment algorithm
                                                                                                                                                                                                                                                                  - 6-week simulation with 3 profiles (improving, struggling, excellent)
